# ⚽ Football Ticket Booking System

#### 📌 Overview

A PostgreSQL database project for managing football match tickets, users, and booking transactions.

#### 🚀 Features

* User Management
* Match Management
* Ticket Booking System
* Payment Status Tracking
* Relational Database Design



#### 🗄️ Database Tables

* **Users**
* **Matches**
* **Bookings**

#### 🔗 Relationships

* One User → Many Bookings
* Many Bookings → One Match
* One Booking → One User & One Match

#### 📝 SQL Concepts Used

* SELECT
* WHERE
* LIKE & ILIKE
* COALESCE
* INNER JOIN
* LEFT JOIN
* Subqueries
* Aggregate Functions
* ORDER BY
* LIMIT & OFFSET

#### 📊 Implemented Queries

* Available Champions League Matches
* User Search by Name
* NULL Value Handling
* Booking Details with JOIN
* Users Without Bookings
* Above Average Booking Cost
* Top Expensive Matches

#### 🛠️ Technologies

* PostgreSQL
* SQL
* Draw.io to make a ERD
